# 顺子功能编号总表

新版编号按大类记：

- `P` = 图片功能
- `V` = 视频功能

使用时不用死记完整编号。能写编号就写编号；不写编号也可以，顺子会根据你的文案自动判断，并在输出里反标编号。

## 使用减负规则

| 你怎么说 | 顺子怎么处理 |
| --- | --- |
| `P01-1` | 精确执行游戏推荐海报 |
| `P01` | 先判断属于哪种社媒图，再输出 |
| `社媒活动开奖图` | 自动识别为 `P01-2` |
| `私域兑换码图` | 自动识别为 `P03-2` |
| `15 秒视频脚本` | 自动识别为 `V01` |

## 图片功能

| 编号 | 功能 | 子类 / 用途 | 输出重点 |
| --- | --- | --- | --- |
| `P01` | 社媒图 | TG / INS / 活动 / 日常海报 | 平台、尺寸、品牌、信息层级 |
| `P01-1` | 游戏推荐海报 | 游戏介绍、玩法推荐、热门游戏 | 游戏卖点、画面冲击、角色/无角色方案 |
| `P01-2` | 活动开奖海报 | 开奖、中奖、bonus、奖励 | 奖励氛围、金额/规则可读、到账感 |
| `P01-3` | 品牌公告海报 | 品牌通知、规则公告、重要提醒 | 清楚、正式、低误读 |
| `P01-4` | 收集引导海报 | 引导关注、提交信息、加入社群 | CTA 明确、步骤清楚 |
| `P01-5` | 特定活动 / 节日海报 | 节日、体育热点、品牌节点 | 节日氛围、品牌融合、不过度堆元素 |
| `P02` | 白底图 | 纯白背景、方便抠图 | 主体集中、边缘清楚、无飞散元素 |
| `P02-1` | 人物白底图 | 单人、半身、全身 | 姿态清楚、边缘干净 |
| `P02-2` | Banner 白底图 | 长条物料、横幅素材 | 横向构图、主体完整 |
| `P02-3` | 人物 + 文案白底图 | 人物搭配短标题 | 人物与文字连接紧凑 |
| `P02-4` | 道具 / 物料白底图 | 金币、奖杯、礼盒、道具 | 材质可信、结构不乱 |
| `P03` | 私域图 | 社群公告、兑换码、贴纸风 | 不出现真人，亲和、清楚、轻博彩 |
| `P03-1` | 私域公告 | 社群通知、活动提醒 | 信息直接、可读性强 |
| `P03-2` | 兑换码图 | code、奖励领取、福利提醒 | 兑换码突出、按钮感强 |
| `P03-3` | 私域贴纸风海报 | 社区贴纸、到账感、奖励感 | 手机截图、贴纸元素、不过重黑金 |
| `P04` | 人物 / KOL 图 | 品牌代言人、KOL、头像、场景图 | 人设稳定、品牌一致 |
| `P04-1` | 新 KOL 设计 | 新建品牌代言人 | 三视图、特写、品牌配饰、性格特征 |
| `P04-2` | 已有 KOL 出图 | 旧 KOL 延展 | 自拍、节日、头像、服装、场景 |
| `P05` | 产品 / 物料图 | 礼盒、奖杯、金币、App、活动物料 | 产品结构、质感、主视觉 |
| `P06` | 文案与标题 | 文案提炼、标题升级、CTA | 标题、副标题、利益点、层级 |
| `P07` | 视觉诊断与质检 | 图片诊断、交付前检查、返工复盘 | 问题、优先修改、可沉淀规则 |

## 视频功能

| 编号 | 功能 | 用途 | 输出重点 |
| --- | --- | --- | --- |
| `V01` | 视频脚本 | 短视频、KOL 视频、口播 | 剧情大概、主旨、分段脚本 |
| `V02` | 分镜图方案 | 给分镜图生成或设计准备 | 16:9 分镜画布、6/8 张画面、数字编号 |
| `V03` | 视频生成提示词 | Seedance / Sora / AI 视频 | 中英文写在一起，方便整段复制 |
| `V04` | 视频整包 | 一次要完整视频方案 | 脚本 + 分镜 + 中英文提示词 + 剪辑注意 |

## 后续新增规则

- 新图片大类用下一个 `P` 编号，例如 `P08`。
- 新视频大类用下一个 `V` 编号，例如 `V05`。
- 新子类追加在大类后面，例如 `P03-4`。
- 已发布编号不要改含义，只能标注 `UPDATED` 或 `PLANNED`。
