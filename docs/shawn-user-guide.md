# 顺子使用说明

这份说明给新人、老同事和负责人一起看。目标是：知道顺子能做什么，知道怎么开口，知道输出结果怎么看。

## 一句话用法

```text
顺子，编号或功能，具体需求。
```

编号可以不写。你写 `P01`、`P01-2`、`V01` 会更准；不写编号时，顺子会根据文案自动判断，并在输出里告诉你它判断的是哪个功能。

## 最常用说法

```text
顺子，P01，帮我给 X89 做一张 TG 游戏推荐海报，1:1。
```

```text
顺子，做一张私域兑换码图。
```

```text
顺子，V01，写一个 15 秒 KOL 视频脚本。
```

```text
顺子，V03，给我一版中英文视频提示词，写在一起方便复制。
```

## 图片功能

### P01 社媒图

适合：TG 图、INS 图、游戏推荐、开奖、公告、收集引导、节日活动。

子类：

- `P01-1` 游戏推荐海报
- `P01-2` 活动开奖海报
- `P01-3` 品牌公告海报
- `P01-4` 收集引导海报
- `P01-5` 特定活动 / 节日海报

顺子会根据不同海报给不同视觉方案，包括配色、构图、透视、有角色 / 无角色方案。

### P02 白底图

适合：纯白背景、方便后期抠图、素材复用。

子类：

- `P02-1` 人物白底图
- `P02-2` Banner 白底图
- `P02-3` 人物 + 文案白底图
- `P02-4` 道具 / 物料白底图

硬规则：纯白背景、主体集中、边缘清楚、没有飞散金币/彩带/粒子/烟雾。

### P03 私域图

适合：社群公告、兑换码、私域贴纸风海报。

子类：

- `P03-1` 私域公告
- `P03-2` 兑换码图
- `P03-3` 私域贴纸风海报

当前默认不出现真人。后续可扩展小 IP、表情包、IP + 海报。

### P04 人物 / KOL 图

适合：品牌代言人、虚拟 KOL、头像、节日图、场景图。

子类：

- `P04-1` 新 KOL 设计：三视图、特写、品牌配饰、性格特征、服装、禁用项。
- `P04-2` 已有 KOL 出图：日常自拍、节日庆祝、头像、不同服装、不同场景。

### P05 产品 / 物料图

适合：礼盒、奖杯、金币、App、活动道具、品牌物料。

顺子会重点控制结构、材质、光影、主体和 logo 风险。

### P06 文案与标题

适合：运营文案太长、标题不够强、CTA 不明确。

顺子会整理主标题、副标题、利益点、补充说明和 CTA。

### P07 视觉诊断与质检

适合：图不好看、不知道哪里怪、交付前检查。

顺子会输出主要问题、优先修改、可保留部分、下一版方向和质检清单。

## 视频功能

### V01 视频脚本

适合：短视频、KOL 视频、口播、活动视频。

输出必须清楚直观，包含剧情大概、主旨、分段脚本和节奏。

### V02 分镜图方案

适合：要做视频分镜图或 AI 分镜参考。

固定规则：

- 分镜图画布是 `16:9`。
- 分镜图里面放目标视频比例的单张画面。
- 一张分镜图至少 6 张。
- 剧情丰富用 8 张。
- 超过 8 张拆成两张分镜图。
- 图里不要文字标注，只保留数字编号。

### V03 视频生成提示词

适合：Seedance、Sora、即梦等视频工具。

输出规则：

- 中英文写在一起。
- 一整段方便复制。
- 包含画面、动作、镜头、节奏、风格、负面要求。

### V04 视频整包

适合：一次要完整视频方案。

输出包括：脚本、分镜图说明、中英文视频提示词、剪辑注意。

## 新人怎么用

先不用记编号。直接说：

```text
顺子，我要做一张 TG 活动开奖图，品牌 X89，尺寸 1:1。
```

顺子会自动判断功能编号。

## 老同事怎么用

尽量带大类编号，减少沟通：

```text
顺子，P01，做 TG 活动图。
```

如果知道子类，可以写更细：

```text
顺子，P01-2，做活动开奖海报。
```

## 维护者怎么提需求

说功能编号 + 要改的问题：

```text
P03-2 兑换码图要增加更强的按钮感和复制码视觉。
```

这样后续 PR、更新说明、测试都能准确定位。
