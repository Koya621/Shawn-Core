# 顺子使用说明

这份说明给新人、老同事和负责人一起看。目标很简单：知道顺子能做什么，知道怎么开口，知道输出结果怎么看。

## 先记住一句话

顺子不是“随便写提示词”的工具。顺子会先帮你把需求拆清楚，再给可以复制使用的结果。

推荐说法：

```text
顺子，功能编号，具体需求。
```

示例：

```text
顺子，F03，帮我给 X89 写一张 TG 活动海报提示词，1:1，主打 bonus。
```

## 常用功能

### F01 需求拆解

适合：需求不完整、怕返工、不知道先问什么。

你可以说：

```text
顺子，F01，帮我看这个需求缺什么。
```

顺子会输出：品牌、平台、尺寸、文案、素材、交付物、缺失信息。

成功标志：你知道下一步要补哪几个信息。

### F02 品牌素材调用

适合：要按某个品牌风格做，不想串品牌。

你可以说：

```text
顺子，F02，按 DREAM77 的风格来，帮我整理注意事项。
```

顺子会输出：品牌风格、常用素材、不能混用的内容、建议方向。

成功标志：输出里明确写出品牌，不会把其他品牌套进来。

### F03 海报 / 社媒提示词

适合：海报、TG 图、INS 图、活动图、游戏宣传图。

你可以说：

```text
顺子，F03，写一版 TOP ONE 的 INS 日常海报提示词，3:4。
```

顺子会输出：设计判断、Prompt、Negative Prompt、后期排版建议、确认点。

成功标志：提示词能直接复制到生图工具或交给设计师继续做。

### F04 白底图 / 抠图素材

适合：要纯白背景、主体完整、方便抠图。

你可以说：

```text
顺子，F04，帮我写一个金币礼盒白底图提示词。
```

顺子会特别避开：飞散金币、彩带、粒子、烟雾、复杂背景。

成功标志：主体集中，边缘清楚，适合后期抠图。

### F05 角色 / KOL 形象

适合：虚拟 KOL、人物海报、人设设计、角色动作。

你可以说：

```text
顺子，F05，帮我做一个品牌 KOL 形象方向，年轻、亲和、适合短视频。
```

顺子会输出：人物定位、服装、动作、镜头、表情、风格风险。

成功标志：人物设定清楚，后续能连续做图和视频。

### F06 产品 / 物料海报

适合：礼盒、奖杯、金币、App 界面、活动奖励、物料包装。

你可以说：

```text
顺子，F06，帮我做一个奖杯和金币的活动物料海报提示词。
```

顺子会重点控制：物体结构、材质、光影、logo 不变形、主体不乱。

成功标志：产品或物料是画面主角，不会被背景抢走。

### F07 文案提取与重组

适合：运营文案太长、重点不清楚。

你可以说：

```text
顺子，F07，把这段活动文案整理成海报标题、副标题和 CTA。
```

顺子会输出：主标题、副标题、利益点、补充说明、CTA。

成功标志：海报上应该放什么字，一眼清楚。

### F08 字体 / 标题升级

适合：标题不够醒目、没有设计感、层级弱。

你可以说：

```text
顺子，F08，这个标题不够有冲击力，帮我升级几个方向。
```

顺子会输出：标题结构、字体气质、描边/投影/立体建议、排版位置。

成功标志：标题更强，但仍然可读。

### F09 风格学习 / 案例学习

适合：看到一张参考图，想让顺子学它的风格。

你可以说：

```text
顺子，F09，学习这张参考图，帮我拆成可复用规则。
```

顺子会输出：保留点、可借鉴点、不能照抄点、提示词方向。

成功标志：下次能复用这个风格，而不是只会模仿一张图。

### F10 视觉诊断

适合：图不好看、不知道哪里怪、返工前想先判断。

你可以说：

```text
顺子，F10，帮我诊断这张图哪里要改。
```

顺子会输出：主要问题、优先修改、可保留部分、下一版方向。

成功标志：知道先改什么，不会越改越乱。

### F11 AI 生图 / 修图

适合：直接生图、局部修图、改背景、改质感、扩展方向。

你可以说：

```text
顺子，F11，按这张参考图生成一版更适合 TG 的活动图。
```

顺子会输出或执行：提示词、负面词、保留内容、后期排版建议。

成功标志：关键素材不乱跑，提示词能稳定复用。

### F12 视频脚本 / 分镜 / 视频提示词

适合：短视频、KOL 视频、AI 视频、口播脚本。

你可以说：

```text
顺子，F12，写一个 15 秒 KOL 视频脚本，适合 Instagram。
```

顺子会输出：视频方向、分段脚本、画面分镜、字幕文案、视频生成 Prompt。

成功标志：剪辑同事或 AI 视频工具能照着做。

### F13 质检复盘

适合：交付前检查、返工后沉淀经验。

你可以说：

```text
顺子，F13，帮我做发布前质检。
```

顺子会检查：品牌、尺寸、文案、logo、主体、裁切、命名、能否交付。

成功标志：交给运营前少出错。

### F14 生图不可用兜底

适合：GPT 当前模式不能直接生图。

你可以说：

```text
顺子，F14，当前不能生图怎么办？
```

顺子会给：可复制 Prompt、切换入口建议、上传参考图提醒。

成功标志：即使不能直接生图，也不会卡住生产。

## 最推荐的使用习惯

- 先说功能编号。
- 再说品牌。
- 再说平台和尺寸。
- 最后说目标和保留素材。

示例：

```text
顺子，F03，品牌 X89，平台 TG，尺寸 1:1，做 bonus 活动海报，保留手机截图和 logo。
```

## 不推荐的说法

```text
帮我弄好看一点。
```

这样太模糊，顺子会先反问很多信息。

更好的说法：

```text
顺子，F10，帮我诊断这张图哪里不好看，优先告诉我最该改的 3 个点。
```
