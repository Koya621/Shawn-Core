## 本版重点更新

- 图片和视频分成 `P / V` 两个大类。
- 子类编号从 `P01.01` 改成 `P01-1`，减少输入负担。
- 顺子支持自动识别，不写编号也能判断。
- 视频分镜图规则固定：16:9 画布、6/8 张画面、只用数字编号。
- TG、公盘、提示词网站增加同步包，不直接改机器人归档逻辑。
