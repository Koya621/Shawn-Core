# awesome-gpt-image-2 接入笔记

来源仓库：

```text
https://github.com/YouMind-OpenLab/awesome-gpt-image-2
```

## 这个库的定位

这是一个大型、持续更新的 ChatGPT Image 提示词库。

它适合作为 Shawn 的外部参考源，不适合作为 Shawn 的长期全文知识库。

## 为什么不能直接塞进 Shawn

- 体量大
- 内容更新频繁
- prompt 类型多且杂
- 全量导入会让 Shawn 变钝、变乱、变得不稳定

## 正确接入方式

Shawn 对这个仓库应采用：

- 按需检索
- 分类提炼
- 结果沉淀

而不是全文吸收。

## 建议提炼的内容类型

### 1. Prompt 结构

重点看：

- 人物海报
- 产品海报
- 字体排版
- 白底图
- 场景氛围图

提炼成：

- `prompt-patterns.md`

### 2. 风格语言

重点看：

- 科技感
- 商业大片
- 潮流视觉
- 极简高级
- 编辑感视觉

提炼成：

- `style-library.md`

### 3. 负面约束写法

重点看：

- 如何避免脸部变形
- 如何限制乱飘元素
- 如何保持产品准确
- 如何提高文字可读性

提炼成：

- `negative-prompt-library.md`

### 4. 控制方法

重点看：

- 参考图锁定方式
- 角色一致性控制
- 配饰和物料对齐方式

提炼成：

- `asset-locking.md`

## 使用方式

以后如果用户想让 Shawn 参考这个库，建议用这种说法：

- 参考 `awesome-gpt-image-2` 里和人物海报相关的写法，帮我提炼
- 从这个库里找适合白底图的 prompt 模式，整理成顺子的规则
- 用这个库里的 typography 或商业海报思路，帮我升级 Shawn 的提示词模板

## 工作原则

Shawn 不对整个库做“记住”。

Shawn 只对目标明确的子集做：

1. 观察
2. 归类
3. 提炼
4. 回写

这样才能持续吸收，而不会失控。
