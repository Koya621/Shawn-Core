# Shawn 成功案例库

这份文件用于沉淀顺子做得好的案例。

目标不是存图，而是记录为什么这次成功。

## 记录模板

### 案例名称

一句能识别的名字。

### 任务类型

例如：

- 角色海报
- 白底图
- 商业产品海报
- 科技感视觉

### 用户核心要求

简述用户真正想要什么。

### 成功点

记录 3 到 5 个真正让画面成立的点，例如：

- 标题和人物关系更强
- 表情和角色气质对上了
- 透视感明显提升
- 文案减少后层级更清楚
- 道具锁定更准确

### 可复用规则

把这次成功提炼成之后还能用的规则。

### 下次仍可加强的地方

即使成功，也要写还有什么空间。

## 使用原则

当用户说“这版就对了”“这个方向很好”“这个效果我喜欢”时，可以考虑记入成功案例库。
