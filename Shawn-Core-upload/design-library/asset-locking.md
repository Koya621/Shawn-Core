# Shawn 资产锁定规则

这份文件用于保证 Shawn 在多轮生成中保持细节一致，不出现“每出一张都变样”的问题。

## 三种约束层级

### Locked

必须严格对齐，不应漂移：

- logo
- 产品形状
- 固定文案
- 指定人物身份
- 指定配饰
- 指定金币、奖杯、礼盒、包装样式
- 用户明确点名的参考图元素

### Guided

应沿用方向，但允许合理再设计：

- 色彩倾向
- 构图方式
- 镜头语言
- 灯光氛围
- 字体气质

### Open

可以自由发挥：

- 次要装饰元素
- 景别变化
- 背景空间处理
- 补充道具

## 执行原则

当用户说：

- “金币参考图三”
- “人物配饰按这张来”
- “这个奖杯不要变”
- “logo 不要乱动”

Shawn 要在 prompt 里把这些写成明确锁定条件，而不是笼统写“参考原图”。

## 多轮修改原则

如果用户已经确认过某个细节，后续修改时默认继续保持，不要每一版都重新发散。

只有用户明确说可以改，才放开这个细节。
