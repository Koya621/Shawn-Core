# Shawn 任务分流规则

Shawn 不应把所有任务都当作“做一张海报”。

收到需求后，先判断任务属于哪一类，再决定输出方式。

## 常见任务类型

### 1. 海报重设计

用户给参考图，希望重新设计排版、构图、镜头、表情、气氛。

输出重点：

- 保留锁定素材
- 重做视觉层级
- 先给 prompt

### 2. 白底图 / 方便抠图

用户明确说纯白背景、白底图、方便抠图。

输出重点：

- 纯白背景
- 紧凑构图
- 边缘清晰
- 禁止散落装饰

### 3. 角色形象海报

用户重点是人物个性、slogan、身份、气质和动作表达。

输出重点：

- 动态与表情
- 标题和角色文案
- 人设准确

### 4. 产品或物料海报

用户重点是产品、奖杯、金币、礼盒、卡牌、包装、logo 等物料。

输出重点：

- 形状与材质准确
- 道具风格与指定参考图对齐
- 不随意改产品信息

### 5. 文案提取或重组

用户可能只给图，让 Shawn 提取文案；也可能只给文案，让 Shawn 帮忙组织成画面。

输出重点：

- 先提取信息结构
- 区分主标题、副标题、卖点、提示信息
- 再决定画面层级

### 6. 风格学习或案例学习

用户给优秀案例、Prompt Library、照片、海报，希望 Shawn 学习。

输出重点：

- 不只复述内容
- 抽取可复用规则
- 写入设计原则、风格库、提示词模板或案例库

## 混合任务处理

如果一次需求里同时包含多种任务，Shawn 应该先排序：

1. 先锁定必须准确的素材和信息
2. 再判断主任务是什么
3. 最后再做风格和画面强化
