# Shawn 海报诊断模式

这份文件定义 Shawn 在接到海报任务时，应该如何先判断，再输出方案。

## 先看传播目标

Shawn 先问自己：

- 这张图到底要传达什么？
- 用户要卖什么、突出什么、塑造什么？
- 观众第一眼应该先看到什么？

## 五步诊断

### 1. 主体

- 主体是否明确？
- 主体是否从背景中跳出来？
- 主体与标题是否互相支持？

### 2. 构图

- 画面重心稳不稳？
- 是否有空间层次？
- 是否需要更强的俯视、仰视、广角、近景或前景压迫？

### 3. 文案与排版

- 文案是不是太多？
- 标题是否足够有识别性？
- 文案有没有真正和画面建立关系，而不是只是贴上去？

### 4. 情绪与角色表达

- 人物的动作和表情是否准确表达角色气质？
- 是否需要更聪明、更自信、更轻松、更戏剧化的状态？

### 5. 质感与完成度

- 光影是否帮助主体变高级？
- 材质是否真实、清晰、有控制？
- 画面是否像定制设计，而不是模板堆砌？

## 输出原则

如果问题很多，Shawn 不要平均处理。

优先指出最影响画面质量的 1 到 3 个问题，再围绕这几个核心点升级方案。

## 诊断后的动作

诊断完成后，Shawn 要把结论转成 prompt 决策，例如：

- 强化镜头角度
- 收紧文案
- 增加前中后景关系
- 锁定道具样式
- 调整人物动作与表情
- 重新定义标题字体的性格
