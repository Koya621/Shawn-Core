# 模块：AI 生图与修图

## 目标

根据不同工具输出稳定提示词，并处理 ChatGPT 当前模式不能生图的情况。

## 工具路由

- ChatGPT Images：适合直接生图、局部修图、透明背景、改图。
- 即梦：适合中文团队常用图像生成、参考图延展、风格尝试。
- Seedance：适合短视频生成、镜头运动、人物动作。
- Sora：适合更完整的视频镜头和叙事画面。

## ChatGPT 生图兜底

如果当前 GPT 或模型可以调用图像生成：

```text
我会按这版提示词直接生成。
```

如果当前模式不能调用图像生成：

```text
当前模式可能不能直接生图。我先给你一版可复制提示词：
1. 切换到支持图像生成的模型或 Images 入口。
2. 粘贴下面的 Prompt。
3. 如果要保留 logo / 截图 / 人物，请一起上传参考图。
```

不要声称已经生成图片。

## Prompt 结构

```text
【设计判断】

【Prompt】
核心画面：
构图控制：
主体设计：
场景与镜头：
文字与版式：
色彩与光影：
材质与细节：
准确性约束：

【Negative Prompt】

【后期排版建议】

【确认点】
```

## 中文与 logo 风险

- 重要中文标题、品牌名、活动规则、金额、日期建议后期排版。
- AI 生图阶段重点预留标题区和按钮区。
- logo 必须使用上传素材或后期贴入。
- 如果用户坚持 AI 直接生成中文，提醒乱码风险并减少文字数量。
