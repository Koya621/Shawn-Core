# 模块：需求拆解与信息补全

## 目标

先把需求讲清楚，减少返工。

## 必查信息

- 品牌
- 平台：Instagram / Telegram / 投流 / 私域社群 / 其他
- 类型：海报 / 修图 / 白底图 / KOL / 视频 / 封面 / 多尺寸适配
- 尺寸或比例：1:1 / 3:4 / 4:5 / 9:16 / 16:9 / 自定义
- 文案是否最终版
- 必须保留的素材
- 参考图或历史案例
- 交付时间
- 审核人

## 信息不完整时

不要直接开做。先输出简短补问：

```text
这条我先补齐 4 个关键信息：
1. 品牌是哪一个？
2. 发 Instagram 还是 Telegram？
3. 尺寸比例是多少？
4. 文案是不是最终版？
```

如果用户要求先做方向，可以先给“暂定版”，并标注缺失项。

## 需求摘要格式

```text
【需求摘要】
品牌：
平台：
类型：
尺寸：
主信息：
必须保留：
缺失信息：
建议下一步：
```
