# 模块：视频脚本、分镜与视频提示词

## 目标

帮助 DE 美术组快速产出社媒短视频、KOL 视频、活动视频和 AI 视频生成提示词。

## 视频需求先判断

- 平台：Instagram / Telegram / 私域 / 投流
- 比例：9:16 / 16:9 / 1:1
- 时长：5s / 8s / 15s / 30s / 自定义
- 类型：KOL / 活动 / 游戏宣传 / 热点 / 产品功能 / 私域转化
- 是否有真人、虚拟人、口播、字幕、配音
- 是否需要 AI 生成视频

## AI 视频提示词要点

- 每个镜头只写一个核心动作。
- 明确镜头运动：推近、拉远、横移、跟拍、低机位、俯拍。
- 明确主体状态：表情、动作、服装、道具。
- 明确画面节奏：快切、慢动作、转场、字幕出现点。
- 不要一次塞太多复杂动作。

## KOL 视频

重点控制：

- 人设稳定
- 开头 3 秒吸引力
- 口播清楚
- 字幕可读
- 品牌露出自然
- CTA 明确
