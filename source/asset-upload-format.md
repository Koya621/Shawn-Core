# 品牌素材上传格式

## 推荐结构

每个品牌一个文件夹：

```text
品牌名/
  brand-card.md
  logo-primary.png
  logo-primary.svg
  examples/
    good-01.png
    good-02.png
    good-03.png
  avoid/
    bad-01.png
  links.md
```

## 必传内容

- `brand-card.md`
- logo PNG 或 SVG
- 3-10 张历史优秀案例

## 可选内容

- 禁用示例
- 可复用背景
- 常用人物
- 常用游戏素材
- 常用按钮 / 奖励 / 金币 / 奖杯

## 不建议直接上传 GPT 的内容

- PSD
- AE / PR 工程
- 长视频
- 超大图片包
- 内部聊天记录
- 账号、密码、后台截图、客户隐私

这些内容放公司盘或 GitHub Release，在 `brand-card.md` 或 `links.md` 里写链接。

## 文件命名

```text
品牌_平台_类型_尺寸_版本_日期
```

示例：

```text
X89_TG_bonus-poster_1x1_v01_20260616.png
TOPONE_INS_kol-video-cover_9x16_v03_20260616.png
```

成功标志：任何同事看到文件名，都能知道这是哪个品牌、哪个平台、什么用途、什么尺寸、哪个版本。
