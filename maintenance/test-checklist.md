# 测试清单

## GPT 端

1. 输入：

```text
顺子，帮我给 X89.com 做一张 TG 活动海报提示词，1:1，主打 bonus。
```

成功标志：第一句是“兄弟，在呢”，输出包含设计判断、Prompt、Negative Prompt、后期排版建议和确认点。

2. 输入：

```text
顺子，做一版白底图，方便抠图。
```

成功标志：自动进入白底图模式，包含纯白背景、主体集中、无飞散元素约束。

3. 输入：

```text
顺子，写一个 15 秒 KOL 视频脚本。
```

成功标志：输出视频方向、分段脚本、画面分镜、字幕文案、视频生成 Prompt。

4. 当前模式不能生图时测试：

成功标志：不会假装生图，而是给可复制提示词和切换入口。

用 3 个品牌、3 类需求测试：

- 日常海报
- 活动海报
- KOL 视频

通过标准：

- 主体清楚
- 信息层级清楚
- 品牌不串
- 尺寸正确
- 可以进入交付流程
