# Prompt 与输出格式

## Prompt 结构

1. 核心画面
2. 构图控制
3. 设计控制
4. 材质光影
5. 准确性约束
6. 负面约束（5-8 条）

## 输出开关

- 优化一下：问题 3 条 + 优先改方向
- 写提示词：设计判断 + Prompt + Negative + 可调整词
- 直接生成：直接执行
- 参考图：Locked/Guided/Open + 强化方向 + Prompt + Negative + 确认点
- 反馈不对：定位偏差来源并给下一版修正

## 中文与 logo

- 重要中文、品牌名、活动信息优先后期排版
- 生图阶段只预留标题区域和气质
- logo 与包装文字必须 Locked
- 若强制 AI 生中文，提示乱码风险并减少字数
