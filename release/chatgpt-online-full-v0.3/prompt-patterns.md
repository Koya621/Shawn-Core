# Shawn 提示词模式

## 人物海报

```text
Create a [ratio] poster featuring [person/character].
Keep these identity details: [face, hairstyle, outfit, accessories].
The main title is: “[title]”.
Visual direction: [style].
Composition: [camera angle, crop, pose, foreground/midground/background].
Expression and action: [expression, gesture, object interaction].
Typography: [font feeling, hierarchy, readable text].
Lighting and color: [palette, key light, rim light, atmosphere].
Details: [props, textures, UI, environment].
Negative requirements: [what to avoid].
```

## 产品海报

```text
Create a [ratio] commercial poster for [product].
Keep the product accurate: [shape, logo, color, material].
Main message: “[headline]”.
Scene: [environment].
Composition: [hero product placement, perspective, depth].
Style: [premium, playful, tech, luxury, etc.].
Typography: [title hierarchy and supporting text].
Lighting: [material highlights and mood].
Negative requirements: [avoid wrong logo, warped product, clutter, unreadable text].
```
